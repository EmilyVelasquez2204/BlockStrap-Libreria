Redondear elementos

Por tamaños

Los tamaños disponibles para redondear son: small, medium, large, xl, xxl.

Para rendondear un elemento debe colocar rde-s (small) o el tamaño que prefiera.

Slime bebé (Redondear de tamaño small)
Cabeza de Creeper (Redondear de tamaño medium)
Bloque de diamante (Redondear de tamaño large)
Cabeza de Ghast (Redondear de tamaño XL)
Sol de Minecraft (Redondear de tamaño XXL)
Fragmentos de código:
<div class = "rde-s p-3 sf-3 m-5 cf-gel color-font-hoja">Slime bebé (Redondear de tamaño small)</div>

<div class = "rde-m p-4 sf-3 m-5 color-font-color-fondo cf-neon ">Cabeza de Creeper (Redondear de tamaño medium)</div>

<div class = "rde-l p-4 sf-3 m-5 cf-diamante">Bloque de diamante (Redondear de tamaño large)</div>

<div class = "rde-xl p-4 sf-4 m-5 color-font-gris-fantasma cf-fantasma">Cabeza de Ghast (Redondear de tamaño XL)</div>

<div class = "rde-xxl p-4 sf-5 m-5 color-font-naranja-lava cf-fuego">Sol de Minecraft (Redondear de tamaño XXL)</div>
