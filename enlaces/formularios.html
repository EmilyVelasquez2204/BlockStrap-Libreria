Combobox o listas

También puedes agregar una lista desplegable para que el usuario elija varias opciones a la vez.


Fragmentos de código:
<form action=" ">

<select size = "1">

<option value="1">Creeper</option>

<option value="2">Golem</option>

<option value="3">Enderman</option>

<option value="4">Zombie</option>

<option value="5">Esqueleto</option>

</select>

</form>


Tamaños

Los comboBox o listas vienen en distintos tamaños: small, medium, large y xl, para que se adapten a tu diseño.

Fragmentos de código:
<select class ="form-Cmb-small" size="1">

<option selected>Tamaño Small </option>

</select>

<select class="form-Cmb-medium" size="1">

<option selected> Tamaño Medium </option>

</select>

<select class = "form-Cmb-large" size="1">

<option selected> Tamaño Large </option>

</select>

<select class = "form-Cmb-xl" size="1">

<option selected> Tamaño XL </option>

</select>

Elementos desactivados

Puedes desactivar cualquier parte del formulario para evitar que el usuario interactúe con ciertos elementos.


Fragmentos de código:
<label> ComboBox desactivado </label>

<select disabled size="1">

<option selected> Criaturas:</option>

</select>

<label> Botón desactivado </label>

<button disabled>Desactive</button>

<label> TextBox Desactivados </label>

<input disabled type="text" aria-placeholder="email">
